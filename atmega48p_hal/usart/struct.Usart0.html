pub fn interrupt_rxc(&mut self, state: bool)[src]

Enable/disable "RX Complete" interrupt

When this interrupt triggers, new data is available to be read from the data-register. The corresponding ISR is USARTi_RX (where i is this peripheral's number). For example, for USART1 on ATmega32U4:

#[avr_device::interrupt(atmega32u4)]
fn USART1_RX() {
    // ...
}

pub fn interrupt_udre(&mut self, state: bool)[src]

Enable/disable "USART Data-Register Empty" interrupt

This interrupt signals that new data can be written to the data-register. The corresponding ISR is USARTi_UDRE (where i is this peripheral's number). For example, for USART1 on ATmega32U4:

#[avr_device::interrupt(atmega32u4)]
fn USART1_UDRE() {
    // ...
}
